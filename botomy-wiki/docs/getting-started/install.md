# Install

## Download the app

Get started by **[downloading](https://botomy.vercel.app) and installing the app**.

Or **try Botomy immediately** in the **[browser](https://botomy.vercel.app/dist/web/index.html)**.

### Supported Platforms

- MacOS
- Windows
- HTML5 (limited support - use at your own risk)
